# Przestrzenna-analiza-wra-liwo-ci-na-skutki-zmian-klimatu-z-wykorzystaniem-danych-BDOT10k-
Repozytorium obejmuje przykład zautomatyzowanej analizy wrażliwości przestrzeni wybranego miasta na skutki zmian klimatu z wykorzystaniem model designer w oprogramowaniu QGIS. Głównym źródłem danych wykorzystanym w analizie jest Baza Danych Obiektów Topograficznych w skali 1:10 000.
